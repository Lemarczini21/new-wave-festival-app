# New Wave Festiwal App
Application for Ticket order for Festiwal.
> Live demo [https://replit.com/@Lemarczini21/furniture-shop](https://replit.com/@Lemarczini21/furniture-shop)


## Technologies Used
- React
- Redux
- CSS/SASS

## Features
- Advanced filters for products
- Changing currency
- RWB
- Swipe functionality
